# Agents CLI — Guide for co-workers {#mainpage}


Welcome to the consolidated documentation for the **Agents CLI** and **Collaborative Work Environments**. This guide is designed to take you from a local setup to a fully collaborative development workflow.

## 1. Getting Started: The Agents CLI

The core of this repository is the **Agents CLI**, a tool for coordinating multiple assistant agents (Copilot, Gemini, Claude, etc.) directly from your terminal. Whether you are automating workflows or experimenting with multi-agent reasoning, this is your entry point.

### Setup & Tools
To begin, you need to install the necessary CLI agents. We have prepared a comprehensive guide that covers:
*   **Installation:** Copilot CLI and Gemini CLI setup.
*   **Concepts:** How multi-agent coordination works.
*   **Best Practices:** Prompting and reasoning patterns.
